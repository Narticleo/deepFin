與黑潮海洋文教基金會合作，將蒐集十幾年的花紋海豚照片做系統化的整理，進行背鰭偵測切割，再以背鰭上的特徵做個體辨識。
個體會隨觀測的結果增加，為此本計畫採用metric learning的方式，將背鰭轉換化為向量，根據相似度判定個體類別，面對新
個體時，只需以向量相似度作為判斷而無須重新訓練模型。
